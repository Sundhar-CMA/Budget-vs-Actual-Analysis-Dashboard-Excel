# Budget-vs-Actual-Analysis-Dashboard-Excel

Budgeting Dashboard – Excel Project
Project Overview

This project is an interactive Budgeting Dashboard built in Microsoft Excel to compare Actual Sales against Budgeted Sales across different inventory types, warehouse locations, and years.

The dashboard enables users to analyze sales performance, identify variances between actual and budgeted figures, and explore product-level insights through dynamic filtering.

Dataset
-Sample dataset provided as part of a guided Excel dashboard project.

Data includes:  
-Sales Revenue  
-Budgeted Sales  
-Inventory Types  
-Product Information  
-Warehouse Locations  
-Year-wise Performance Data

Dashboard Features  
-Interactive Filters  
-Year Slicer   
-Inventory Type Slicer

Performance Analysis    
-Actual vs Budgeted Sales by Inventory Type    
-Actual vs Budgeted Sales Trend by Year    
-Actual vs Budgeted Sales by Warehouse Location    
-Top 5 Products by Sales Revenue

Dynamic Reporting    
-Dashboard visuals automatically update based on selected slicers.    
-Top 5 products section updates dynamically according to selected inventory type and year.
